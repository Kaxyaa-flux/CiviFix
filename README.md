## 🌟 Problem Statement

Traditional civic complaint systems suffer from several challenges:

* ❌ Lack of transparency
* ❌ Fake or duplicate reports
* ❌ No immutable proof of submitted complaints
* ❌ Slow communication between citizens and authorities
* ❌ Poor tracking and accountability

CiviFix addresses these problems by combining Artificial Intelligence, Blockchain technology, and real-time updates into one unified platform.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/CiviFix.git
```

```bash
cd CiviFix
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Start Backend

```bash
npm run server
```

---

# 📌 Environment Variables

Create a `.env` file and add:

```env
DATABASE_URL=
JWT_SECRET=
GEMINI_API_KEY=
BLOCKCHAIN_RPC_URL=
PORT=5000
```
